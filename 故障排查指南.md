## ✅ 检查清单 1：域名解析

### 1.1 测试域名解析
在你的本地电脑打开 **命令提示符（CMD）** 或 **PowerShell**，执行：

```bash
ping 2048.hnwen17.top
```

**可能的结果：**

#### ✅ 成功情况：
```
正在 Ping 2048.hnwen17.top [123.456.789.10] 具有 32 字节的数据:
来自 123.456.789.10 的回复: 字节=32 时间=20ms TTL=54
```
→ 如果显示你的服务器IP，说明解析正常，跳到 **检查清单2**

#### ❌ 失败情况：
```
Ping 请求找不到主机 2048.hnwen17.top。请检查该名称，然后重试。
```
→ 说明域名解析未生效，请检查：

**解决方案：**
1. 登录腾讯云控制台 → 云解析DNS
2. 确认已添加记录：
   - 主机记录：`2048`
   - 记录类型：`A`
   - 记录值：你的服务器公网IP（不是127.0.0.1）
3. 如果刚添加，等待10-30分钟
4. 如果已等待很久，删除记录重新添加

### 1.2 确认服务器IP
在腾讯云控制台 → 云服务器 → 实例列表 → 查看**公网IP**
- ✅ 正确：类似 `123.45.67.89`
- ❌ 错误：`127.0.0.1` 或 `192.168.x.x`（这是内网IP）

## ✅ 检查清单 2：服务器安全组和防火墙

### 2.1 腾讯云安全组（重要！）
1. 登录腾讯云控制台
2. 进入 **云服务器** → 找到你的实例
3. 点击实例ID → **安全组** 选项卡
4. 点击安全组ID进入规则配置
5. 检查 **入站规则** 是否包含：

| 协议 | 端口 | 来源 | 策略 |
|-----|------|------|------|
| TCP | 80 | 0.0.0.0/0 | 允许 |
| TCP | 443 | 0.0.0.0/0 | 允许 |
| TCP | 8888 | 0.0.0.0/0 | 允许（宝塔面板端口）|

**如果没有，点击"添加规则"添加以上端口。**

### 2.2 宝塔面板安全设置
1. 登录宝塔面板
2. 点击左侧 **安全** 菜单
3. 检查放行的端口列表中是否包含：
   - `80`（HTTP）
   - `443`（HTTPS）
4. 如果没有，点击"放行端口"添加

### 2.3 测试端口是否开放
在本地电脑浏览器访问：
```
http://你的服务器IP
```

**可能的结果：**
- ✅ 显示宝塔默认页面/其他网站 → 端口开放正常，跳到 **检查清单3**
- ❌ 无法访问/连接超时 → 端口未开放，返回2.1和2.2检查

## ✅ 检查清单 3：宝塔网站配置

### 3.1 检查网站是否创建成功
1. 登录宝塔面板
2. 点击左侧 **网站** 菜单
3. 确认列表中有 `2048.hnwen17.top`
4. 查看状态是否为 **运行中**

### 3.2 检查网站目录和文件
1. 在网站列表中，点击 `2048.hnwen17.top` 右侧的 **根目录**
2. 或者点击 **文件** → 进入 `/www/wwwroot/2048.hnwen17.top`
3. 确认目录中有且仅有以下文件：
   ```
   index.html
   script.js
   style.css
   ```
4. 确认没有多余的文件（如404.html、index_backup.html等）

### 3.3 检查文件权限
在文件列表中，选中所有文件，右键点击 → **权限** → 设置为 `755`

### 3.4 查看网站配置文件
1. 在网站列表中，点击站点右侧的 **设置**
2. 点击 **配置文件** 选项卡
3. 检查配置中的关键部分：

```nginx
server {
    listen 80;
    server_name 2048.hnwen17.top;
    index index.html;
    root /www/wwwroot/2048.hnwen17.top;

    # 其他配置...
}
```

确认：
- `server_name` 是 `2048.hnwen17.top`
- `root` 是 `/www/wwwroot/2048.hnwen17.top`
- `index` 包含 `index.html`

## 🔧 常见问题快速解决

### 问题1：显示"无法访问此网站"或"连接超时"
**原因：** 端口未开放
**解决：** 检查腾讯云安全组和宝塔安全设置（检查清单2）

### 问题2：显示"403 Forbidden"
**原因：** 文件权限问题或缺少index.html
**解决：**
- 检查文件权限设为755
- 确认index.html存在且文件名正确（区分大小写）

### 问题3：显示"404 Not Found"
**原因：** 网站根目录配置错误
**解决：** 检查宝塔网站配置文件中的root路径

### 问题4：显示"502 Bad Gateway"
**原因：** PHP配置错误（纯静态不需要PHP）
**解决：** 网站设置 → PHP版本 → 选择"纯静态"

### 问题5：ping不通域名
**原因：** DNS解析未生效
**解决：**
- 检查腾讯云DNS记录是否正确添加
- 等待解析生效（最多24小时）
- 尝试刷新本地DNS缓存：`ipconfig /flushdns`（Windows）
