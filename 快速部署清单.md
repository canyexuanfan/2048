# 2048.hnwen17.top 快速部署清单 ✅

## 📌 第一步：域名解析（5分钟）

### 腾讯云DNS解析设置：
1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 进入 **云解析 DNS** 服务
3. 找到 `hnwen17.top` 域名，点击解析
4. 添加以下记录：

| 主机记录 | 记录类型 | 记录值 | 说明 |
|---------|---------|-------|------|
| 2048 | A | 你的服务器公网IP | 指向腾讯云服务器 |

5. 保存后等待5-10分钟生效

### 验证解析：
```bash
ping 2048.hnwen17.top
```
返回你的服务器IP即为成功！

---

## 🚀 第二步：宝塔面板配置（3分钟）

### 2.1 登录宝塔
- 访问：`http://你的服务器IP:8888`
- 输入账号密码

### 2.2 创建网站
1. 左侧菜单 → **网站** → **添加站点**
2. 填写：
   - **域名：** `2048.hnwen17.top`
   - **根目录：** `/www/wwwroot/2048.hnwen17.top`
   - **FTP：** 不创建
   - **数据库：** 不创建
   - **PHP版本：** 纯静态
3. 点击提交

### 2.3 上传文件
1. 点击 **文件** 菜单
2. 进入目录：`/www/wwwroot/2048.hnwen17.top`
3. 删除默认文件（index.html、404.html等）
4. 点击 **上传** 按钮
5. 上传以下3个文件：
   - ✅ `index.html`
   - ✅ `script.js`
   - ✅ `style.css`
   - ❌ 不要上传 `server.py`

### 2.4 配置SSL证书（推荐）
1. 网站列表中找到 `2048.hnwen17.top`
2. 点击右侧 **设置**
3. 选择 **SSL** 选项卡
4. 选择 **Let's Encrypt**
5. 勾选域名，点击 **申请**
6. 申请成功后，开启 **强制HTTPS**

---

## 🎮 第三步：访问测试

打开浏览器访问：
- HTTP：`http://2048.hnwen17.top`
- HTTPS：`https://2048.hnwen17.top`（配置SSL后）

能看到游戏界面并正常玩耍即部署成功！

## ⚠️ 常见问题

### 域名访问不了？
- ✓ 检查DNS解析是否生效（ping测试）
- ✓ 检查腾讯云安全组：开放80、443端口
- ✓ 检查宝塔安全设置：开放80、443端口

### SSL证书申请失败？
- ✓ 确保域名已解析到服务器
- ✓ 确保80端口可访问（Let's Encrypt验证需要）
- ✓ 等待域名解析完全生效后再申请

### 页面显示403？
- ✓ 检查文件权限：755
- ✓ 确保index.html存在

### 游戏不能玩？
- ✓ 按F12查看浏览器控制台错误
- ✓ 确认3个文件都已正确上传
