# 2048游戏 - 宝塔Linux静态部署指南

## 📋 部署信息

**目标域名：** `2048.hnwen17.top`
**主域名：** `hnwen17.top`
**部署方式：** 纯静态（Nginx）

---

## ⚠️ 第一步：域名解析配置（必须先做！）

### 在腾讯云DNS解析中添加记录：

1. 登录腾讯云控制台
2. 进入 **"云解析 DNS"** 服务
3. 找到域名 `hnwen17.top`，点击 **"解析"**
4. 点击 **"添加记录"**
5. 填写以下信息：

```
主机记录：2048
记录类型：A
线路类型：默认
记录值：你的腾讯云服务器公网IP（例如：123.456.789.10）
TTL：600（默认）
```

6. 点击 **"保存"**
7. 等待5-10分钟生效

### ✅ 验证解析是否生效
在本地电脑打开命令行，执行：
```bash
ping 2048.hnwen17.top
```
如果返回你的服务器IP，说明解析成功！

## 🚀 宝塔面板部署步骤（超简单）

### 第一步：登录宝塔面板
1. 浏览器访问：`http://你的服务器IP:8888`
2. 输入用户名和密码登录

### 第二步：创建网站
1. 点击左侧菜单 **"网站"**
2. 点击 **"添加站点"**
3. 填写信息：
   ```
   域名：2048.hnwen17.top
   根目录：/www/wwwroot/2048.hnwen17.top
   FTP：不创建
   数据库：不创建
   PHP版本：纯静态（选择"纯静态"或不选PHP版本）
   ```
4. 点击 **"提交"**

**注意：** 这不会影响你的主站 `hnwen17.top`，2048游戏会独立运行在子域名上。

### 第三步：上传项目文件
有两种方式上传：

#### 方式1：直接上传（推荐）
1. 在宝塔面板，点击 **"文件"** 菜单
2. 进入目录：`/www/wwwroot/2048.hnwen17.top`
3. 删除默认的 `index.html`、`404.html` 等文件
4. 点击 **"上传"** 按钮
5. 选择并上传这3个文件：
   - `index.html`
   - `script.js`
   - `style.css`

**重要：不要上传 `server.py` 文件！**

#### 方式2：使用Git
如果你的项目已推送到Git仓库：
1. SSH连接服务器
2. 执行命令：
   ```bash
   cd /www/wwwroot/2048.hnwen17.top
   rm -rf *  # 清空目录
   git clone 你的仓库地址 .
   # 如果用Git方式，server.py在服务器上不会影响运行
   ```

### 第四步：配置SSL证书（启用HTTPS）
1. 在 **"网站"** 列表中，找到刚创建的站点
2. 点击站点名称右侧的 **"设置"**
3. 点击 **"SSL"** 选项卡
4. 选择 **"Let's Encrypt"**
5. 勾选你的域名
6. 点击 **"申请"** 按钮（自动申请免费证书）
7. 申请成功后，开启 **"强制HTTPS"**

### 第五步：访问网站
- HTTP访问：`http://2048.hnwen17.top`
- HTTPS访问：`https://2048.hnwen17.top`（配置SSL后）

🎮 打开浏览器，输入域名即可开始游戏！

## ⚙️ 高级配置（可选）

### Nginx优化配置
如果需要优化性能，可以在宝塔面板的站点设置中，点击 **"配置文件"**，添加以下配置：

```nginx
# 在 server 块中添加以下内容

# 开启Gzip压缩
gzip on;
gzip_vary on;
gzip_min_length 1024;
gzip_types text/css text/javascript application/javascript application/json;

# 静态资源缓存
location ~* \.(js|css|html)$ {
    expires 7d;
    add_header Cache-Control "public, immutable";
}

# 防止访问隐藏文件
location ~ /\. {
    deny all;
}
```

### 防火墙配置
确保以下端口已开放：
- 宝塔面板：点击 **"安全"** → 开放端口 `80`（HTTP）和 `443`（HTTPS）
- 腾讯云安全组：在云控制台添加入站规则，开放 `80` 和 `443` 端口

## 🔧 常见问题排查

### 1. 域名访问不了？
- 检查域名解析是否生效（可能需要等待10分钟）
- 检查腾讯云安全组是否开放80/443端口
- 检查宝塔面板安全设置是否开放80/443端口

### 2. SSL证书申请失败？
- 确保域名已正确解析到服务器IP
- 确保80端口可以正常访问
- 检查域名是否已在其他地方申请过证书（有数量限制）

### 3. 页面显示403 Forbidden？
- 检查文件权限：在宝塔文件管理中，右键点击文件夹，选择"权限"，设置为755
- 确保index.html文件存在且可读

### 4. 游戏无法正常运行？
- 按F12打开浏览器控制台，查看是否有JavaScript错误
- 确保所有文件（index.html, script.js, style.css）都已正确上传
